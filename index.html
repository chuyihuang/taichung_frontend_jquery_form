js基本語法: 針對本次教學

  • 宣告變數
  • 邏輯判斷
  • 加減乘除
  • function

宣告變數

  • 使用關鍵字var: 例如var price, qty...
  • 數字不可在前。例如: name99與99name
  • 命名的慣例: price, itemPrice
  • 切記別取x,y或abc123之類意義不明的名稱

邏輯判斷

  • true與false
  • 使用if (true or false) {...}
  • 使用 else if {...} 與 else {...}

加減乘除

  • + - * / %
  • 數字加數字=數字: 3 + 5 = 8
  • 文字加數字=文字: "3" + 3 = "33"
  • 文字(數字)減數字=數字: "3" - 1 = 2
  • 文字(文字)減文字=NaN
  • Console範例

Function 函數

  • 將重複使用的程式碼獨立出來
  • 使用關鍵字function，將程式碼放在{...}
  • function abc() {...}
  • var abc = function() {...} 與上一行相同
  • 使用方式: abc(arg) //arg為參數
